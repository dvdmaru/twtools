什麼是旅圖？

旅圖 TripCraft 是 TWTools 旗下的 AI 旅遊規劃工具。

只需輸入目的地、日期與預算，旅圖就會用 AI 即時產出個人化的完整行程表，支援全球目的地。

🚀 開發者：dvdmaru

4 步驟開始規劃

使用旅圖非常簡單：

1 選擇目的地

輸入你想去的城市，或從熱門選項中挑選

2 設定日期與預算

選擇出發/回程日、幣別、消費偏好

3 客製化風格

選擇消費等級、節奏、興趣偏好

4 AI 生成行程

一鍵產出完整行程表、地圖、訂購連結

功能亮點

🗺️ 互動式地圖標示每個景點位置
🔄 不滿意？可以「重新規劃某一天」
📅 匯出 .ics 日曆檔或 PDF
✈️ 一鍵搜尋機票、住宿、景點 Maps 連結
🌐 支援繁中、簡中、English
💰 消費等級參考（無魔幻金額）

免費使用

每月可免費使用旅圖 3 次（用 Sonnet 模型生成完整行程）。使用免費額度後，可輸入自己的 Claude API Key 繼續使用。

想了解 API Key 隱私？請查看隱私權政策。
