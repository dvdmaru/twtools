旅圖 TripCraft

隱私權政策

最後更新：2026 年 3 月 18 日

簡介

旅圖 TripCraft（以下稱「本服務」）致力於保護用戶的隱私。本政策說明我們如何蒐集、使用和保護您的資料。

1. 我們不收集個人資料

旅圖不要求您建立帳戶、輸入姓名、電郵或任何個人可識別資訊。您可以完全匿名使用本服務。

2. 裝置識別碼

為了計算免費額度（每月 3 次），我們在您的瀏覽器 localStorage 中存放一個隨機裝置識別碼。

  • 此識別碼不與任何個人資訊關聯
  • 僅用於追蹤該裝置本月的免費使用次數
  • 您可隨時清除瀏覽器 cookies 和 localStorage 重置此識別碼
  • 不會傳送至任何第三方伺服器

3. 行程資料傳送至 AI 服務

當您按下「開始規劃」時，您輸入的旅行資訊（目的地、日期、預算偏好等）會傳送至 Anthropic Claude API 以生成行程。

  • 我們不儲存您的行程資料在我們的伺服器上
  • 資料交互遵循 Anthropic 的隱私政策（anthropic.com/privacy）
  • 生成完成後，數據不會被我們保留

4. API Key 隱私

如果您選擇輸入自己的 Claude API Key 以使用免費額度之外的服務：

  • 您的 API Key僅在瀏覽器端使用
  • 我們的伺服器不會看到您的 API Key
  • 請勿在公共電腦上輸入 API Key

5. 分析工具 (Google Analytics 4)

我們使用 Google Analytics 4 蒐集匿名使用統計，包括：

  • 頁面瀏覽次數
  • 功能使用頻率（如匯出 PDF、搜尋機票等）
  • 設備類型與瀏覽器

重要：Google Analytics 不追蹤個人身份或行程內容。所有資料經過匿名化處理。

6. 廣告與 Google AdSense

我們使用 Google AdSense 在網頁上展示廣告。

  • Google 可能使用 Cookie 提供個人化廣告
  • 您可在 Google 廣告設定中管理廣告偏好
  • 廣告展示與您的行程內容無關

7. 第三方服務連結

旅圖行程中包含的以下連結會導向 Google 服務，受 Google 隱私政策管轄：

  • Google Flights - 機票搜尋
  • Google Hotels - 住宿搜尋
  • Google Maps - 景點與餐廳位置

這些連結包含搜尋參數（目的地、日期），但不傳送任何您的個人資訊。

8. 資料安全

我們採取適當的技術措施保護傳輸中的資料（HTTPS 加密）。然而，互聯網傳輸無法保證 100% 安全。

9. Cookie 與本地存儲

旅圖使用以下瀏覽器存儲機制：

  • localStorage - 儲存裝置識別碼與偏好設定
  • Google Analytics Cookie - 匿名流量分析
  • Google AdSense Cookie - 廣告投放

您可在瀏覽器設定中清除所有 Cookie 與 localStorage。

10. 政策更新

如果我們更新本隱私權政策，將在本頁面公佈新版本。重大變更會通過網站公告通知。

11. 聯絡方式

如有隱私相關問題，請通過問題回報表單聯絡我們。
